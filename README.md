# Course Recommendation System

## 📌 Overview
This project is a machine learning-based course recommendation system using KMeans clustering.

## 🚀 Features
- User clustering using KMeans
- Course recommendation logic
- Streamlit web app

## 📊 Dataset
- User dataset (CSV)
- Synthetic data used for courses and transactions

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
